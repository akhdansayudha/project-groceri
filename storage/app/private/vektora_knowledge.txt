=== 1. TENTANG VEKTORA & MATA UANG ===
Vektora adalah platform Creative Agency modern yang menghubungkan klien dengan talenta profesional (Staff) untuk kebutuhan Desain Grafis, Web Development, dan layanan kreatif lainnya secara on-demand.

[SISTEM TORATIX]
- Definisi: Toratix (TX) adalah mata uang digital eksklusif di Vektora.
- Nilai: Token ini digunakan untuk membayar semua layanan di platform.
- Masa Aktif: LIFETIME (Seumur Hidup). Tidak akan hangus/expired selama akun aktif.
- Cara Kerja: Token dikunci (Locked) saat project dimulai, dan baru ditransfer ke Staff saat project selesai (Completed).

[LAYANAN DI VEKTORA]
Di Vektora menawarkan layanan berikut:
1. Logo Design
2. UI/UX Landing Page
3. Motion Graphic (20s)
4. Brand Design
5. UI/UX Mobile App
6. Illustration
7. Graphic Design
- untuk melihat daftar harga tiap layanan, silahkan cek di menu Pricing/bisa juga langsung cek saat membuat Request.

=== 2. MEMBERSHIP TIERS (LOYALTY PROGRAM) ===
Status Membership didapatkan secara otomatis berdasarkan "Total Lifetime Purchased" (Akumulasi pembelian Token sejak akun dibuat). Status TIDAK AKAN TURUN.

[A] STARTER MEMBER
- Syarat: Total pembelian 0 - 49 Token.
- Benefit:
  1. Akses ke seluruh layanan dasar Vektora.
  2. Akses fitur Workspace & Manajemen Project standar.
  3. Support via Email & Tiket.
  4. Maksimal tasks active = 1
  5. Maksimal Workspace = 1

[B] PROFESSIONAL MEMBER
- Syarat: Total pembelian 50 - 199 Token.
- Benefit:
  1. Priority Fast Track: Project Kakak akan diprioritaskan dalam antrian pengerjaan (lebih cepat diambil Staff).
  2. Diskon Spesial: Harga layanan tertentu lebih murah (diskon token).
  3. Slot Project Lebih Banyak: Bisa menjalankan lebih banyak project 'Active' secara bersamaan.
  4. Badge khusus 'Professional' di profil.
  5. Maksimal tasks active = 3
  6. Maksimal Workspace = 3

[C] ULTIMATE MEMBER
- Syarat: Total pembelian 200+ Token.
- Benefit:
  1. Akses VVIP: Layanan prioritas tertinggi.
  2. Senior Talent Access: Project dikerjakan oleh Staff Senior/Expert kami.
  3. Revisi Lebih Longgar: Kebijakan revisi yang lebih fleksibel.
  4. Max Slot Project: Limit project aktif maksimal.
  5. Jalur komunikasi prioritas dengan Super Admin.
  6. Maksimal tasks active = 5
  7. Maksimal Workspace = 10

=== 3. PANDUAN PENGGUNAAN (HOW-TO) ===

[CARA TOP UP TOKEN]
1. Masuk ke menu "My Wallet" di sidebar kiri.
2. Klik tombol "Top Up".
3. Pilih paket Token yang diinginkan.
4. Lakukan pembayaran sesuai instruksi Invoice (Transfer Bank/E-Wallet).
5. Saldo akan masuk otomatis setelah pembayaran terverifikasi sistem.

[CARA MEMBUAT PROJECT BARU]
1. Pastikan saldo Token mencukupi.
2. Klik menu "My Requests" -> "New Request".
3. Pilih "Workspace" (Folder kerja) project tersebut.
4. Pilih "Service Category" (Misal: Logo Design, Web Dev).
5. Isi "Brief Project" dengan detail (Judul, Deskripsi, Deadline, Lampiran File).
6. Klik "Submit Request". Token akan otomatis ter-lock.

[CARA MEMBUAT WORKSPACE]
Workspace adalah folder untuk mengelompokkan project (Misal: "Bisnis Kopi", "Tugas Kuliah").
1. Klik menu "Workspace".
2. Klik "Create New Workspace".
3. Beri nama dan ikon yang sesuai.

[CARA UBAH PASSWORD]
1. Klik menu 'Settings'
2. Dibagian Security & Password, isi password lama dan password baru
3. Klik 'Save Changes'.

[CARA UBAH PROFILE] (Dihalaman Settings ada kolom nama, email, password)
1. Klik menu 'Settings'
2. Dibagian Personal Information, ubah Full Name.  atau ubah bagian Email Address jika ingin ubah email.
3. Klik 'Save Changes'.
